# ログイン画面モック - 説明

目的: デザイナー/PM/開発チームでレイアウトやUXを早期に確認するための静的モックです。

確認ポイント:
- レイアウト: カード幅、余白、ヘッダーとの関係
- レスポンシブ: 320px〜768pxでの表示確認
- フォーカス順: `identifier` → `password` → `ログイン` ボタン（キーボードでTab操作）
- 入力補完: `autocomplete` 属性の適切性
- テキスト・リンク: 「パスワードを忘れた場合」など文言の日本語表現

使い方:
1. ブラウザで `app/pages/login/モック/ログイン画面モック.html` を開く
2. 各表示サイズで確認し、改善点をコメントしてください

推奨アクション:
- 色/スペーシング/ボタンサイズの微調整
- 実装時は `LoginForm.tsx` の `label` と `aria` をモックに合わせる
